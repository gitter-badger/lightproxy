# 上不了网 ERR_EMPTY_RESPONSE

部分用户在 Chrome 中开启代理后会无法访问网络，提示 ERR_EMPTY_RESPONSE 。

可能是由于 /etc/hosts 中缺少默认规则

127.0.0.1	localhost
255.255.255.255	broadcasthost
::1             localhost

这往往可能是 iHosts 等 hosts 修改软件造成的，请打开 hosts 修改软件并应用以上规则。
