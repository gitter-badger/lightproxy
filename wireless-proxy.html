# 无线端代理

LightProxy 也可以支持无线端的代理抓包和转发，点击右侧 手机代理 的 Tab

可以看到一个二维码（不同机器二维码不同的，需要扫描自己的二维码），扫描二维码后下载对应的证书，并且安装。

# 安装证书

安装证书对于不同的手机有不同的操作

# iOS

#

打开相机扫描二维码，在 Safari 中打开，安装描述文件

下载后去 设置 中找到 已下载的描述文件

然后安装描述文件

安装完还有最后一步，到 `设置 => 通用 => 关于本机 => （拉到最下面）证书信任设置，勾上 LightProxy 的证书

# 设置代理

在 Wifi 设置中的 设置代理为界面中显示的地址以及端口号

# 查看请求

点击右侧的 网络请求 能够看到具体的网络请求（不一定是来自无线的，本机的请求也会代理过来）
